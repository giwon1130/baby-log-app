# 컴포넌트 인벤토리

> **누가 봄**: 디자이너 / FE
> **언제 봄**: 새 화면 만들 때 "이미 있는 거 재사용 가능?" 확인

코드 파일 위치 같이 표기. UI 변경 시 여기도 갱신.

## 화면 (`src/screens/`)

| 파일 | 역할 |
|------|------|
| `HomeScreen.tsx` | 빠른 기록 토글 + 오늘 요약 + 다음 수유 시각 |
| `LogScreen.tsx` | 기록 탭 진입점 — 수유/기저귀/수면 서브 화면으로 라우팅 (얇음) |
| `FeedLogScreen.tsx` | 수유 상세 기록 (날짜 필터, 수정, 삭제) |
| `DiaperLogScreen.tsx` | 기저귀 동상 |
| `SleepScreen.tsx` | 수면 동상 |
| `GrowthRecordScreen.tsx` | 성장 기록 입력/리스트 |
| `StatsScreen.tsx` | 주간 차트 |
| `BabyProfileScreen.tsx` | 아기 프로필, 수유 가이드, 알림 설정, 가족 초대 코드 |
| `FamilySetupScreen.tsx` | 첫 실행 — 가족 생성/합류 |
| `CryMonitorScreen.tsx` | 울음 분석 진입점 (녹음 + 결과 표시) |
| `CryHistoryScreen.tsx` | 울음 분석 기록 목록 + 통계 |

## 재사용 컴포넌트 (`src/components/`)

| 파일 | 용도 | Props |
|------|------|-------|
| `cry/ResultView.tsx` | 울음 분석 결과 렌더 (카드 + 디테일 + 피드백) | `sample`, `features`, `onCorrect`, `onConfirm` |
| `cry/AcousticDetails.tsx` | 음향 feature 디테일 카드 | `features` |
| `cry/ConfidenceBar.tsx` | 0–1 confidence 가로 바 | `value` |
| `cry/CorrectionModal.tsx` | 라벨 6종 선택 바텀시트 | `visible`, `onClose`, `onPick` |
| `cry/LearningStageBanner.tsx` | 학습 단계 표시 배너 | `stage` |

> 다른 도메인용 공통 컴포넌트는 아직 추출 안 됨. 같은 패턴이 3+ 화면에서 반복되면 추출 검토.

## 공통 패턴 (인라인 — 추출 안 함)

자주 보이는 UI 블록들. 추출하지 않는 이유: 화면마다 살짝 다름. 비슷하면 컴포넌트.

### 카드 (Card)
```tsx
<View style={{
  backgroundColor: '#fff',
  borderRadius: 14,
  padding: 16,
  gap: 10,
}}>
```

### 히어로 카드 (큰 중앙 카드)
```tsx
<View style={{
  backgroundColor: '#fff',
  borderRadius: 16,
  padding: 24,
  alignItems: 'center',
  gap: 10,
  minHeight: 220,
  justifyContent: 'center',
}}>
```

### Primary CTA 버튼
```tsx
<TouchableOpacity style={{
  flexDirection: 'row', alignItems: 'center', justifyContent: 'center',
  gap: 8, height: 54, borderRadius: 14,
  backgroundColor: '#FF6B9D',
}}>
  <Ionicons name="..." size={22} color="#fff" />
  <Text style={{ color: '#fff', fontWeight: '700', fontSize: 16 }}>...</Text>
</TouchableOpacity>
```

### Link 스타일 버튼 (보조)
```tsx
<TouchableOpacity style={{
  flexDirection: 'row', alignItems: 'center', justifyContent: 'center',
  gap: 6, paddingVertical: 12,
}}>
  <Text style={{ color: '#FF6B9D', fontWeight: '600', fontSize: 14 }}>...</Text>
</TouchableOpacity>
```

### 빈 상태
```tsx
<View style={{
  backgroundColor: '#fff', borderRadius: 14,
  padding: 32, alignItems: 'center', gap: 8,
}}>
  <Ionicons name="time-outline" size={40} color="#FFB7CE" />
  <Text style={{ fontSize: 14, color: '#888' }}>아직 ...이 없어요</Text>
</View>
```

### 배지 (상태 표시)
```tsx
<View style={{
  flexDirection: 'row', alignItems: 'center', gap: 4,
  paddingHorizontal: 8, paddingVertical: 3,
  borderRadius: 10,
  backgroundColor: '<상태별>',
}}>
  <Ionicons size={12} ... />
  <Text style={{ fontSize: 11, fontWeight: '600' }}>...</Text>
</View>
```

배지 컬러: [`디자인시스템.md`](디자인시스템.md) 의 "의미 컬러" 표 참고.

## 네비게이션 구조

```
NavigationContainer
└─ Stack.Navigator (initialRoute: FamilySetup or Main)
   ├─ FamilySetup     # 헤더 없음
   ├─ Main → Tab.Navigator
   │   ├─ Home
   │   ├─ Log → 내부에서 Feed/Diaper/Sleep 분기
   │   ├─ CryMonitor
   │   ├─ Stats
   │   └─ BabyProfile
   ├─ GrowthRecord    # 헤더 있음 (Stack 푸시)
   └─ CryHistory      # 헤더 있음 (Stack 푸시)
```
