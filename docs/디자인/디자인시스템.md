# 디자인 시스템

> **누가 봄**: 디자이너 / FE
> **언제 봄**: 새 화면 만들 때, 색/간격 결정할 때
> **상태**: 코드에서 추출 — 디자인 도구 미사용 (피그마 X). 변경 시 여기 + 코드 동기화.

## 간격

8pt 그리드. 자주 쓰는 값:
- `4` — 인라인 갭 (아이콘-텍스트)
- `8` — 작은 갭
- `10–12` — 카드 안 row 간 간격
- `16` — 카드 padding 기본 / 화면 좌우 padding
- `20–24` — 큰 카드 padding (`heroCard`)
- `32` — 빈 상태 padding

## 모서리 (Border Radius)

| 토큰 | 값 | 용도 |
|------|---|------|

| `circle` | `40` | 펄스 링 |

## 애니메이션

거의 없음. 모달은 RN 기본 `slide`. 카운트다운은 setInterval 기반.

향후: 펄스 링 진짜 펄스 애니메이션 (현재 정적), 카드 진입 fade-in.

## 다크 모드

미지원. 모든 배경 흰색-핑크 톤이라 다크 모드 할 거면 별도 토큰 세트 + 스토어 필요.
1차 출시 후 사용자 요청 보고 결정.

## SoT

각 화면/컴포넌트의 styles는 해당 파일에 인라인. 토큰 추출 안 됨 — 작은 프로젝트라
번거로움 ↑ 기대 효과 ↓. 색상이 5개 화면 이상에서 반복되면 토큰화 검토.
