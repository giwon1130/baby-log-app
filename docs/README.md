# 📚 베이비로그 (앱) 문서

이 폴더는 **사람용** 문서야. 자동화 에이전트(Claude 등) 용 노트는 저장소 루트의
[`AGENTS.md`](../AGENTS.md) 에 있어.

베이비로그는 두 저장소로 나뉘어 있고, 이쪽은 **클라이언트(Expo/React Native 앱)** 저장소야.
백엔드 문서는 [`baby-log-api/docs/`](https://github.com/giwon1130/baby-log-api/tree/main/docs).

## SoT 정책 (어느 쪽에 있는지 헷갈릴 때)

- **제품/디자인/UX/사용자 시나리오** → 이쪽 (앱) 저장소
- **API 명세 / 데이터 모델 / 배포 절차 / 환경변수** → API 저장소
- 양쪽이 같은 토픽 다룰 땐 cross-link
